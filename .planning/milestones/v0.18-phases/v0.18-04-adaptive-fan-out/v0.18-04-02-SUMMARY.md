## Deliverables

### 1. mapRiskLevelToCount Helper Function

**Location:** `hooks/qgsd-prompt.js` (lines 115-122)

**Implementation:**
```javascript
function mapRiskLevelToCount(riskLevel, maxSize) {
  if (riskLevel === 'routine' || riskLevel === 'low') return Math.min(2, maxSize);
  if (riskLevel === 'medium') return Math.min(3, maxSize);
  // 'high', undefined, null, invalid string → fail-open to maxSize
  return maxSize;
}
```

**Behavior:**
- `routine` or `low` → 2 (Claude + 1 external: minimum R3.5 valid quorum)
- `medium` → 3 (Claude + 2 external: balanced quorum)
- `high`, `undefined`, `null`, invalid → maxSize (fail-open to full pool)
- Respects config.quorum.maxSize ceiling via Math.min()

**Exported for testing:** Yes, via module.exports block at end of file

**Covers requirements:** FAN-01 (routine=2), FAN-02 (medium=3), FAN-03 (high=max)
