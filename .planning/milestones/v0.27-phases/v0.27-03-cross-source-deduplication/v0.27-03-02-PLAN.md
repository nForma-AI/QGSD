deduplicateEntries(entries, options = {}):

  - Record merges in mergeLog
- Return { entries, mergeCount, mergeLog }

linkFormalRefs(entries, options = {}):
- requirementsPath = options.requirementsPath || '.formal/requirements.json'
- specDir = options.specDir || '.formal/spec'
- Load requirements JSON (fail-open: if missing, skip auto-detect layer)
- List spec module directories (fail-open: if missing, skip spec-inferred layer)
- For each entry:
  1. If formal_ref is non-null AND formal_ref_source is not null: skip (already linked manually)
     Set formal_ref_source = "manual" if formal_ref is non-null and formal_ref_source is undefined
  2. Auto-detect: tokenize entry.title into lowercase keywords (split on spaces, hyphens, underscores)
     For each requirement: tokenize requirement.text and requirement.id
     Score = count of matching keywords. If score > 0, set formal_ref = "requirement:{id}", formal_ref_source = "auto-detect"
     If multiple matches, use highest score. On tie, use first match.
  3. Spec-inferred: for each module in specDir, check if module name (lowercase) is substring of title (lowercase)
     If match: set formal_ref = "spec:{module}", formal_ref_source = "spec-inferred"
  4. Unmatched: formal_ref = null, formal_ref_source = null
- Return { entries, linkedCount, linkLog }

CommonJS module.exports: { linkFormalRefs }
