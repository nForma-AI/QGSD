## E2E Flow: Bug Discovery → Fix Verification

```
Bug Discovered
  → /nf:debug Step A.5 (formal consultation via debug-formal-context.cjs)
    → formal-scope-scan.cjs --bug-mode → verdict
  → Step E: FORMAL row in result table (reproduced/not-reproduced/no-model)
  → Step G: Gap persistence to bug-model-gaps.json (when no-model)
  → /nf:solve periodic sweep
    → sweepBtoF() classifies: not_covered / covered_not_reproduced / covered_reproduced
    → Dispatches: close-formal-gaps (creation) or model-driven-fix (refinement)
      → 6-phase cycle: discovery → reproduction → refinement → constraints → fix → pre-verify
      → Phase 5b: 2-hop neighbor verification via resolve-proximity-neighbors + run-formal-verify --scope
      → Phase 5c: persist-post-fix-verification → bug-model-gaps.json enrichment
```
