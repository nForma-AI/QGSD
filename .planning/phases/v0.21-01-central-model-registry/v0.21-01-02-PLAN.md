Create `bin/initialize-model-registry.cjs` using Node.js built-in `fs` and `path` modules only (no external npm dependencies).

**Script behavior:**
1. Resolve project ROOT as `path.join(__dirname, '..')`.
2. Check if `formal/model-registry.json` exists. If yes, exit 0 silently (idempotent guard).
3. Scan these subdirectories for model files:
   - `formal/tla/` for `*.tla` files (excluding `*_TTrace_*.tla` files — those are TLC error trace artifacts, not canonical specs)
   - `formal/alloy/` for `*.als` files
   - `formal/prism/` for `*.pm` files
4. For each file found, create a registry entry:
   ```js
   const key = path.relative(ROOT, absPath).replace(/\\/g, '/'); // normalize separators
   // Validate: key must NOT start with './' or '/'
   const isMtime = fs.statSync(absPath).mtime.toISOString();
   const isGenerated = path.basename(absPath).includes('xstate');
   entry = {
     version: 1,
     last_updated: isMtime,
     update_source: isGenerated ? 'generate' : 'manual',
     source_id: isGenerated ? 'generate:tla-from-xstate' : null,
     session_id: null,
     description: ''
   };
   ```
5. Build the registry object:
   ```json
   {
     "version": "1.0",
     "last_sync": "<ISO timestamp now>",
     "models": { "<relative/path>": { ...entry } }
   }
6. Write to `formal/model-registry.json` using `JSON.stringify(registry, null, 2)` with keys sorted for consistent diffs. Sort using `Object.keys(registry.models).sort()` to rebuild models in order.
7. Write stdout: `[initialize-model-registry] Created formal/model-registry.json with N entries`.

**Key constraints from research:**
- Registry keys MUST use `path.relative(process.cwd(), absolutePath)` — validate no `./` or `/` prefix by asserting `!key.startsWith('/')` and `!key.startsWith('./')`. Strip leading `./` if present.
- Use `formal/` as the subdirectory anchor, not absolute paths.
- TTrace files (`*_TTrace_*.tla`) are NOT canonical specs — skip them explicitly.
- `.cfg` files are TLC configs, not specs — skip them (only `.tla`, `.als`, `.pm`).

Add a registry update helper inside `generate-formal-specs.cjs` and call it after each `atomicWrite()` call.

**Add this helper function near the top of the file (after ROOT/DRY definitions):**
```js
function updateModelRegistry(absPath) {
  if (DRY) return; // dry-run: skip registry update
  const registryPath = path.join(ROOT, 'formal', 'model-registry.json');
  if (!fs.existsSync(registryPath)) return; // registry not initialized yet — skip silently
  const registry = JSON.parse(fs.readFileSync(registryPath, 'utf8'));
  const key = path.relative(ROOT, absPath).replace(/\\/g, '/');
  const now = new Date().toISOString();
  const existing = registry.models[key] || {};
  registry.models[key] = {
    version: (existing.version || 0) + 1,
    last_updated: now,
    update_source: 'generate',
    source_id: 'generate:formal-specs',
    session_id: null,
    description: existing.description || ''
  };
  registry.last_sync = now;
  // Atomic write: tmp file + rename (matches existing pattern in this file)
  const tmpPath = registryPath + '.tmp.' + Date.now() + '.' + Math.random().toString(36).slice(2);
  fs.writeFileSync(tmpPath, JSON.stringify(registry, null, 2), 'utf8');
  fs.renameSync(tmpPath, registryPath);
}
```
