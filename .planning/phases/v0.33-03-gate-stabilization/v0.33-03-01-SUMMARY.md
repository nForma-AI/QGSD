# Plan 01 Summary: Gate-Stability Module

## What Was Built
Created `bin/gate-stability.cjs` — a standalone CommonJS module providing flip-flop detection and cooldown enforcement for the gate promotion pipeline.

### Exported Functions
- **countDirectionChanges(entries)** — Counts direction alternations in changelog entries; handles duplicate entries without inflation (Pitfall 1)
- **detectFlipFlops(changelog, threshold)** — Groups entries by model, flags models with >= threshold direction changes
- **isCooldownSatisfied(stabilityInfo)** — Checks both session count AND wall-time conditions before allowing re-promotion
- **updateCooldownState(existingStability, currentGateScore, gateThreshold)** — Increments stable session counter or resets to 0 on regression (Pitfall 3)
- **createUnstableEntry(directionChanges)** — Creates fresh UNSTABLE stability object with initialized cooldown

### Exported Constants
- LEVEL_ORDER, DEFAULT_FLIP_FLOP_THRESHOLD (3), DEFAULT_REQUIRED_SESSIONS (3), DEFAULT_REQUIRED_WALL_TIME_MS (3600000)
