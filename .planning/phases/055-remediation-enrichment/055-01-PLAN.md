In section 3a (R->F Gaps) of commands/nf/solve-remediate.md, insert a coderlm seed-file discovery step BEFORE the existing close-formal-gaps dispatch.

After extracting the uncovered requirement IDs and applying cascade budget truncation, add:

**Coderlm seed-file discovery (CREM-01):**

```
**Coderlm seed-file discovery (CREM-01):** Before dispatching close-formal-gaps, attempt to collect implementation and caller files for the batch of uncovered requirements. This is fail-open — if coderlm is unavailable or returns errors, proceed without seed files.

Run the following node snippet to collect seed files via coderlm's synchronous API:
```bash
node -e "
const _nfBin = (n) => { const p = require('path').join(require('os').homedir(), '.claude/nf-bin', n); return require('fs').existsSync(p) ? p : './bin/' + n; };
const { createAdapter } = require(_nfBin('coderlm-adapter.cjs'));
const adapter = createAdapter({ enabled: true });

// Health check — skip entirely if server not responding
const health = adapter.healthSync();
if (health.error) {
  process.stdout.write(JSON.stringify({ seed_files: [], skipped: true, reason: health.error }));
  process.exit(0);
}

const ids = process.argv.slice(1);
const seedSet = new Set();

for (const id of ids) {
  // getImplementation: find the source file for the requirement's primary symbol
  // Use the requirement ID as the symbol name heuristic (best-effort)
  const impl = adapter.getImplementationSync(id);
  if (impl && impl.file && !impl.error) {
    seedSet.add(impl.file);
    // getCallers: find files that call this implementation
    const callers = adapter.getCallersSync(id, impl.file);
    if (callers && callers.callers && !callers.error) {
      for (const c of callers.callers.slice(0, 3)) {
        if (c.file) seedSet.add(c.file);
      }
    }
  }
}

process.stdout.write(JSON.stringify({ seed_files: [...seedSet], skipped: false }));
" -- ${IDS_COMMA_SEPARATED//,/ } 2>/dev/null || echo '{"seed_files":[],"skipped":true,"reason":"node error"}'
```

Parse the JSON output. If `seed_files` is non-empty, set `SEED_FILES_ARG="--seed-files=$(IFS=,; echo "${seed_files[*]}")"`. Otherwise set `SEED_FILES_ARG=""`.

Log: `"R->F: coderlm seed discovery: {N} seed files for {M} requirements (skipped={skipped})"`
```

Then update the dispatch commands to append `$SEED_FILES_ARG`:

Change:
```
/nf:close-formal-gaps --batch --ids=REQ-01,REQ-02,...
```
To:
```
/nf:close-formal-gaps --batch --ids=REQ-01,REQ-02,... $SEED_FILES_ARG
```

And change:
```

```
To:
```
/nf:close-formal-gaps --batch --all $SEED_FILES_ARG
```

Note: `$SEED_FILES_ARG` is empty string when coderlm is unavailable — this preserves the existing dispatch behavior exactly (fail-open, CADP-02).
