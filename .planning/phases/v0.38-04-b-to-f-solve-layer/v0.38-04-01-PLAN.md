```javascript
function sweepBtoF() {
  if (fastMode) {
    return { residual: -1, detail: { skipped: true, reason: 'fast mode' } };
  }
  // ... classification logic
}
```

**Classification algorithm:**

a. **Get failing tests from t_to_c:** Access the t_to_c result from the residual computation. The t_to_c sweep stores its fail list in `t_to_c.detail.failures` or `t_to_c.detail.failedTests`. If t_to_c was skipped (residual -1) or has no failures, return `{ residual: 0, detail: { total_failing: 0, ... } }`.

b. **For each failing test, classify:**
   - Call `loadBugModelGaps()` from formal-scope-scan.cjs (or inline the load from `.planning/formal/bug-model-gaps.json`)
   - Load `generate-traceability-matrix.cjs --json` output (or reuse if already computed in sweepRtoF)
   - For each failing test file path:
     1. Find linked requirement IDs via traceability matrix (test file → requirement mapping)
     2. If no requirements linked → classification: `not_covered` (+1)
     3. If requirements found, check model-registry.json for formal models covering those requirements
     4. If no models found → classification: `not_covered` (+1)
     5. If models found, check bug-model-gaps.json for reproduction status:
        - If status `reproduced` for any matching bug_id → `covered_reproduced` (0)
        - Otherwise → `covered_not_reproduced` (+1)

c. **Compute residual:** `not_covered_count + covered_not_reproduced_count`

d. **Build detail object:**
```javascript
{
  total_failing: totalFailingTests,
  covered_reproduced: coveredReproducedCount,
  covered_not_reproduced: coveredNotReproducedCount,
  not_covered: notCoveredCount,
  top_bugs: topBugIds.slice(0, 5),  // Top 5 bug IDs for reporting
  classification: classifications,    // Array of { test, classification, models, bug_id }
}
```

e. **Helper functions** (add within nf-solve.cjs, near the sweep functions):
   - `classifyFailingTest(testPath, traceMatrix, modelRegistry, bugGaps)` — returns `{ classification, models, bug_id }`
   - Use `require('crypto').createHash('sha256').update(testPath).digest('hex').slice(0, 8)` for deterministic bug_id (matching v0.38-01-01 decision)

f. **Error handling:** Wrap in try/catch. If any dependency (traceability matrix, model registry, bug-model-gaps) fails to load, return `{ residual: -1, detail: { error: true, stderr: message } }`. This is fail-open — a broken B->F sweep does not block the rest of the solver.
