## TLA+ Properties Verified by TLC

- **SafetyInvariant1**: `decision = "BLOCK" => hasCommand` — no false blocks on non-planning turns
- **SafetyInvariant2**: `decision = "BLOCK" => ~hasQuorumEvidence` — never blocks when quorum satisfied
- **SafetyInvariant3**: `(decision = "PASS" /\ hasCommand) => hasQuorumEvidence` — planning turns require quorum to pass
- **LivenessProperty1**: `<>algorithmDone` — algorithm always terminates
- **LivenessProperty2**: `hasQuorumEvidence => <>(decision = "PASS")` — quorum satisfied → eventually passes
- **LivenessProperty3**: `(hasCommand /\ ~hasQuorumEvidence) => <>(decision = "BLOCK")` — planning without quorum → eventually blocked

## Decisions

- **Fairness in Spec formula**: WF_vars() declared inside QGSDStopHook.tla's Spec, not in MCStopHook.cfg. Matches MCoscillation.cfg pattern — prevents TLC liveness errors from conflicting declarations.
- **detectLivenessProperties integration**: Runner uses run-tlc.cjs's detectLivenessProperties for inconclusive detection (mirrors run-oscillation-tlc.cjs pattern).
