#### 3. Quorum Diagnostic Injection (Plan 03)

Refinement loop → diagnostic generation:
```
model passed (INCOMPLETE outcome)
→ generateDiagnosticFeedback(output, bugTraceJsonPath, bugContext)
→ returns {mismatch_diff, correction_proposals, trace_alignment}
→ stores in iteration.diagnostic
→ fires onDiagnosticGenerated callback
```

Callback → quorum dispatch:
```
model-driven-fix workflow captures diagnostic from callback
→ JSON.stringify(diagnostic)
→ passes as: quorum-slot-dispatch --review-context '<diagnostic-json>'
→ buildModeAPrompt detects JSON.parse succeeds + mismatch_diff && correction_proposals present
→ calls formatDiagnosticForPrompt(parsed)
→ sets reviewContext = null (prevent double-render)
→ injects "## Model Diagnostic Feedback" section into prompt
→ worker receives: mismatch diff + 3 correction proposals ranked by priority
```

- Assertion: Raw JSON string NOT present in final prompt
