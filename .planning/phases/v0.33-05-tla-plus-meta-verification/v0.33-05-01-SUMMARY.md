# Plan 01 Summary: NFSolveConvergence TLA+ Spec

## What was built

Created the NFSolveConvergence.tla formal specification modeling the outer nf:solve loop with:

- **Cross-session state**: session counter, per-layer residuals with history, oscillation counts
- **Option C blocking**: blocks layer immediately upon first oscillation detection (>= 1 predicate, diverging from research Pattern 2's > 1 to satisfy safety invariant)
- **Gate maturity transitions**: ADVISORY -> SOFT_GATE -> HARD_GATE (forward-only)
- **Cascade-aware grace periods**: ActivateGrace correctly iterates downstream layers (layers whose DownstreamOf set contains the improving layer)
- **Safety invariant**: OscillationBounded -- no layer oscillates more than once
- **Liveness property**: EventualConvergence -- solve loop eventually converges under weak fairness
