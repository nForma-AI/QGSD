**Before starting:** Read formal/diff-report.md to determine which fix direction to take.

**If dominant recommendation is IMPL-BUG (most likely based on research)**:

The research identifies the most probable cause as: `validate-traces.cjs` creates a FRESH ACTOR per event (line 224 of existing code), which means multi-event context interactions are invisible. The "69% divergence" is likely a structural artifact of single-step validation — the machine's guards depend on context that only builds up across multiple events in a session, but the validator tests each event from scratch (always starting in IDLE).

Fix: Update `validate-traces.cjs` to use SESSION-based replay (maintain actor state across events from the same session/round_id). Group conformance events by `round_id` (already present in event schema), replay each group through a single actor, and only count divergences within the session context.

Implementation in `bin/validate-traces.cjs`:
1. After parsing all lines, group events by `event.round_id` (or `event.session_id`). Events without `round_id` are standalone.
2. For each round group, create ONE actor, replay all events in sequence.
3. Check state after LAST event in group vs expected final state (DECIDED).
4. A round is divergent only if the final state doesn't match outcome.

```javascript
// Group events by round_id for session-based replay
const rounds = new Map(); // round_id → events[]
for (const event of parsedEvents) {
  const key = event.round_id || event.session_id || 'standalone-' + Math.random();
  if (!rounds.has(key)) rounds.set(key, []);
  rounds.get(key).push(event);
}

// Replay each round through a single actor
for (const [roundId, roundEvents] of rounds) {
  const actor = createActor(qgsdWorkflowMachine);
  actor.start();
  let roundDiverged = false;
  for (const event of roundEvents) {
    const xstateEvt = mapToXStateEvent(event);
    if (!xstateEvt) continue; // skip unmappable (counted separately)
    actor.send(xstateEvt);
  }
  const finalSnapshot = actor.getSnapshot();
  actor.stop();
  // A round passes if it reached DECIDED (any outcome) or stays in a valid state
  if (!finalSnapshot.matches('DECIDED') && !finalSnapshot.matches('IDLE')) {
    divergences.push({ round_id: roundId, finalState: finalSnapshot.value, reason: 'session_incomplete' });
    roundDiverged = true;
  }
  if (!roundDiverged) valid++;
}
```
