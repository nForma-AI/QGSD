    STEP B — Refactor queryEdgesSync in bin/nf-solve.cjs (near line 158):

    Current approach: queries getCallersSync(basename, script) to find which files call a layer script, then does string matching to map caller files back to layer keys. This is file-level, not symbol-level.

    New approach using getImplementation for symbol precision:
    1. For each active layer key with a known script in LAYER_SCRIPT_MAP, derive the main export symbol name (convention: same as the basename without .cjs, camelCase — e.g., 'formal-scope-scan.cjs' -> 'formalScopeScan', or use the exact function name if known from LAYER_SCRIPT_MAP)

    3. The response's callers array contains { file, line } objects — map each caller's file path back to a layer key using LAYER_SCRIPT_MAP reverse lookup (exact path match, not substring)
    4. Build edges: { from: callerLayerKey, to: targetLayerKey }

    If getImplementationSync returns { error } for a symbol, skip it (fail-open: that layer contributes no edges).
    Wrap the entire for-loop body in try/catch returning empty edges on any exception (CADP-02 compliance).

    Add a new LAYER_SYMBOL_MAP constant below LAYER_SCRIPT_MAP that maps each layer key to its primary exported function name. Derive names from the existing scripts — e.g.:

    ```
    const LAYER_SYMBOL_MAP = {
    ```

    ```
    function reindex(opts) {
      try {
        opts = opts || {};
        const port = opts.port || DEFAULT_PORT;
        // POST to /reindex endpoint (or GET /reindex — match coderlm server API)
        // Use the same healthCheckSync spawnSync pattern to make a synchronous HTTP request
        // coderlm server reindex endpoint: POST http://localhost:{port}/reindex
        // Returns { ok: true } on success, { error } on failure
        const result = httpPostSync('http://localhost:' + port + '/reindex');
        if (result.status && result.status >= 200 && result.status < 300) {
          return { ok: true };
        }
        return { error: 'reindex failed: status ' + result.status };
      } catch (e) {
        return { error: e.message };  // fail-open
      }
    }
    ```
