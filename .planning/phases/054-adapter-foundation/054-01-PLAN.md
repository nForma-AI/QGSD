<feature>
  <name>LRU Cache for coderlm Query Results (CADP-01)</name>
  <files>bin/coderlm-cache.cjs, bin/coderlm-cache.test.cjs</files>
  <behavior>
    createLRUCache(capacity, ttlMs) returns a cache object with:
      get(key) -> value | undefined  (undefined on miss or expired entry)
      set(key, value) -> void  (evicts LRU if at capacity; always updates access time)
      reset() -> void  (clears all entries and resets stats)
      stats() -> { hits, misses, size, hitRate }  (hitRate = hits / (hits + misses), 0 if no queries)

    Default values: capacity=100, ttlMs=300000 (5 min)
    Key generation: callers use JSON.stringify({method, symbol, file}) as key

    Cases:
      get on fresh key -> undefined (miss, misses++)
      set then get within TTL -> value (hit, hits++)
      set then get after TTL -> undefined (miss, misses++)
      set 101 entries -> entry 1 (LRU) is evicted, entries 2-101 remain
      get entry 2, then set 100 more -> entry 2 is NOT evicted (it was recently used), entry 3 is evicted
      reset() -> stats() returns {hits:0, misses:0, size:0, hitRate:0}
  </behavior>
  <implementation>
    Use a Map for O(1) lookup + a doubly-linked list for LRU ordering (classic LRU pattern).
    Alternatively: Map + manual promotion on access (simpler: on get/set, delete then re-insert to maintain insertion order = access order in a Map).
    Map insertion-order trick: for LRU with Map, delete+re-insert on access gives O(1) eviction of map.keys().next() (least recently accessed).
    Each entry: { value, expiresAt: Date.now() + ttlMs }.
    On get: if entry found but expiresAt &lt;= Date.now(), delete from map, return undefined (miss).
    CommonJS: 'use strict', module.exports = { createLRUCache }.
    No external dependencies.
  </implementation>
</feature>

    Create bin/coderlm-cache.cjs with 'use strict' and CommonJS exports.

    Implement createLRUCache(capacity=100, ttlMs=300000):
    - Internal Map `_cache` (Map insertion order = LRU order via delete+re-insert on access)
    - Counters: _hits=0, _misses=0
    - get(key):
        if key not in _cache: _misses++; return undefined
        entry = _cache.get(key)
        if Date.now() >= entry.expiresAt: _cache.delete(key); _misses++; return undefined
        _cache.delete(key); _cache.set(key, entry);  // promote to MRU
        _hits++; return entry.value
    - set(key, value):
        if _cache.has(key): _cache.delete(key)  // remove for re-insertion (refresh position)
        else if _cache.size >= capacity: _cache.delete(_cache.keys().next().value)  // evict LRU
        _cache.set(key, { value, expiresAt: Date.now() + ttlMs })
    - reset():
        _cache.clear(); _hits = 0; _misses = 0
    - stats():
        const total = _hits + _misses
        return { hits: _hits, misses: _misses, size: _cache.size, hitRate: total === 0 ? 0 : _hits / total }

    module.exports = { createLRUCache }
