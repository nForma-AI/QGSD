  <files>bin/repowise/inject-cochange-debug.cjs</files>
  <action>
Create `bin/repowise/inject-cochange-debug.cjs`:

1. `#!/usr/bin/env node`, `'use strict'`
2. Imports: `{ computeCoChange, getPartnersForFile }` from `./cochange.cjs`, `fs`, `path`
3. `injectCoChangeDebug(filePath, projectRoot, options)`:
   - Runs computeCoChange(projectRoot, options)
   - Calls getPartnersForFile(filePath, result)
   - Returns formatted string: `"CO-CHANGE PARTNERS: The following files frequently change together with ${filePath}:\n${partners.map(p => `  - ${p.partner} (shared commits: ${p.shared_commits}, coupling: ${p.coupling_degree})`).join('\n')}\nConsider investigating these files when debugging ${filePath}."`
   - Returns null if no partners found
4. Cache: uses `.planning/repowise/cochange-cache.json`
5. CLI: `--file=path`, `--project-root`, `--json`, `--help`
6. Exports: `{ injectCoChangeDebug }`
  </action>
