<task type="auto">
  <name>Task 1: Create extract-precedents.cjs extraction tool</name>
  <files>bin/extract-precedents.cjs</files>
  <action>
Create `bin/extract-precedents.cjs` as a Node.js CLI tool following the pattern of `bin/extract-annotations.cjs`.

**CLI interface:**
- Usage: `node bin/extract-precedents.cjs [debates-dir] [output-path]`
- Defaults: debates-dir = `.planning/quorum/debates/`, output-path = `.planning/quorum/precedents.json`
- Exit 0 on success with summary to stderr (e.g., "Extracted 42 precedents (27 skipped: 15 INCONCLUSIVE, 8 stale, 4 malformed)")

**Core functions (all exported via module.exports):**

1. `extractPrecedentMetadata(filePath, content)` — Parse a single debate markdown file:
   - Extract `Question:` line (case-insensitive, whitespace-tolerant regex)
   - Extract `Date:` as YYYY-MM-DD
   - Extract `Consensus:` as APPROVE/BLOCK/INCONCLUSIVE (case-insensitive)
   - Extract `## Outcome` section content (everything until next `## ` or EOF)
   - Return `{ question, date, consensus, outcome, source_file, computed_at }` or null if required fields missing
   - Skip INCONCLUSIVE consensus entries (only APPROVE and BLOCK are precedents)
   - Log skipped files with reason to stderr (missing field, INCONCLUSIVE, etc.)

2. `isPrecedentFresh(precedent, maxAgeMs)` — TTL check:
   - Default maxAgeMs = 90 * 24 * 60 * 60 * 1000 (90 days)
   - Compare precedent.date against current time
   - Return boolean. Handle invalid date strings gracefully (return false + warn)

3. `main(debatesDir, outputPath)` — Orchestrator:
   - Scan debatesDir for `*.md` files
   - Extract metadata from each, skip nulls and INCONCLUSIVE
   - Filter by freshness (isPrecedentFresh)
   - Write `{ precedents: [...], extracted_at, debate_count, skipped_count }` to outputPath
   - Use `JSON.stringify(obj, null, 2)` for readable output

**Regex robustness (per research pitfall 1):**
- Use `\s*` between label and value for whitespace tolerance
- Use `/i` flag for case-insensitive matching on Consensus
- Test against real debate file format (see context file for structure)

**Entry point guard:** `if (require.main === module) { main(...).catch(...) }` pattern.
