## Problem

The repo now has multiple packaged skills, but there is still no dedicated "using-agent-skills" guide explaining:

- what a packaged skill is in nForma
- when to use a packaged skill vs a slash command
- which lifecycle stage each packaged skill belongs to

Without that guide, the catalog grows faster than discoverability.

## Solution

Create a concise meta-guide that explains packaged skills in nForma and shows the intended lifecycle:

`idea-refine -> task-intake -> planning/execution -> verify-work -> shipping-and-launch`

It should stay short, repo-specific, and linked from the main contributor docs.
